How to use Citation and Definition Tags

The both tags are semantic in nature and their print/result will be italic.

The citation tag tells about the reference from where a piece of information has been extracted.

sometimes, cite tag is used to highlight the title of a continuous paragraph.

The definition tag simply defines a new term.

cite Tag

Think and Grow Rich by Napoleon Hill has sold over hundred million copies worldwide.

Fast Works Organization is an imaginary company.

dfn Tag

A definition is a HTML element used to indicate the defining instance of a new term.
