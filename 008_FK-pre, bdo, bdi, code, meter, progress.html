How to use bdo (Bi-directional Override) Tag

This tag can reverse the direction of text.

It has an attribute 'dir', which has two values.

Its default value is ltr (left to right). Other value is rtl (right to left).

Meter Property

This property is used to show the quantities, like temperature etc.

It has five attributes namely law, high, min, max and value.

Attribute 'value' determine how this property will work. It ranges from 0 to 1.
