Create a Sample Table

A table represents data in grid form. It consists of rows and columns.

Speaking simply, the table is a collection of rows and columns.

Every block of grid is known as 'table cell'. In HTML, a table is created row-wise.

To create a table, we have table tag, within which, we have three following tags:

  • Table Heading (th)
  • Table Row (tr)
  • Table Data (td)

Table Heading (th) may be used for row heading as well as column heading.

HTML has 'caption' tag to provide captions for tables.

Before proceeding further, it would be appropriate if we understand two concepts.

The term 'padding' provides space within a cell, i.e. the space between border and text.

The term 'spacing' provides space outside a cell, i.e. the space between the cells of a table.

In this topic, we are about to learn this technique.
