Image Types

Image formation is based on resolution and involves two technologies, i.e. Bitmap and Vector.

Bitmap are the small pixels (dots) with the help of which an image is created.

For an image, standard resolution is 72ppi. In case of Newspapers and Magazines, standard resolution is 130dpi.

It has three formats. Joint Photographer (JPG) format contains a variety of colours and is mostly used for static pictures.

Usually, PNG format does not have background and the utilization of FIG format is for animated items.

Vector images are made up of lines and colours.

This type of images are considered better, as on enlargement, they sustain their quality.

Inserting Image

In HTML, we use 'img' tag to insert an image. It has three attributes namely src, alt and title.

'src' attribute requires the full path of the image.

'alt' attribute is used to provide description of image. It is displayed when image is not visible due to any reason. It should be relevant to the image.

'title' attribute gives message when we place curser on the image.

Note that three attribute relating to image has been abundant by HTML5, namely align (left, right), width and height.

Picture Tag

We can change our picture as per screen size with this tag.

This tag has two inline tags, i.e. 'source' and 'src'.

Inline tag 'source' has two attributes, namely 'media' and 'srcset'.

Attribute 'media' is used to provide responsive size of an image or picture.

Attribute 'srcset' requires the path of that image or picture.

Figure and Figcaption Tag

These tags are introduced by HTML5 for the first time.

Figure tag is used to make a group of images.

Figcaption tag is used within 'figure tag' and provides caption to the images.
