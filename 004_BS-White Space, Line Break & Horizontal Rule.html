What is White Space

Sometimes, programmers use extra blank space between two words or break a paragraph, just to enable them to understand the text or code easily. These spaces are called 'White Space'. Like-wise, we have done this by breaking this very paragraph into three lines.

Usually, the browser ignores this White Space and displays the text without it.

What is a Line Break

Line break defines a single line break between two items.

It does not add or remove anything from the content.

It is an empty element, therefore, needs not to be closed.

For example,
When it rains:
water comes,
if more rain continue, then more water comes.

What is Horizontal Rule

Horizontal Rule defines thematic change in content and is an empty element.

It is used to create a horizontal line between two topics.

Look carefully, we have already applied this between each topic above.
