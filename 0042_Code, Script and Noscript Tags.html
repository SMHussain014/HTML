Code Tag

This tag is used to display code of a programming language.

If this tag is not used, the code may be get executed. To understand, see the following example.

This is a sample of JavaScript
var a = 10; var b = 20; function xyz(){ print(a + b); }; xyz();

Script and Noscript Tags

Script tag is used to create a link between HTML and JavaScript.

It can be used in 'head' tag and 'body' tag

We have two techniques to use script tag, i.e. in-page script amd external script.

It has four attributes, namely type, src, async and defer.

Attribute 'type' tells the browser what type of programming language is to be used.

Attribute 'src' provides the path from where a programming file exists.

Attributes 'async' and 'defer' provides the method how to execute the programming language.

If method 'async' is used, the web-site will be loaded first and then the programming language.

The method 'defer' reverse the above process.

In case, our browser does not support a programming language, then noscript tag will be executed.
