How to create Scalable Vector Graphs with or without Canvas

Vector graphs can be created in HTML by using 'svg' tag.

It is of three types/shapes, i.e. circle, rect and poly

To create circle, we need x-axis (cx), y-axis (cy) & radius (r) values.

To create rectangular, we need x, y, width and height values.

To create polygon, we need x, y and z values.

To make-up them, we may use CSS.







Map and Area Elements

We can use a shape to work as hyperlink.

For that we use 'image' element. Inside it, attribute 'src' is used to call the image. Thereafter, we use another attribute 'usemap' to assign a title to that image.

By the use of 'map' element, we call with attribute 'name' that image. The value of which is same as of 'usemap' attribute.

Thereafter, we defines the area of that image by using 'area' element.

Element 'area' has two attributes, i.e. 'shape' attribute (its value is the type of svg) and 'coords' attribute which provides its co-ordinates.
