Types of Code in HTML5

There are two types of codes which are in use in HTML5, i.e. Structural Code and Semantic Code.

Structural codes are conventional codes that we are using previously. However, to cope with emerging technology of now-a-days, Semantic codes have also been invented so that disabled persons, specially blind, may also use web.

Semantic code works like conventional structural code and caste no effect on web-site, but they enables Search Engines and Screen Readers to help out disabled person.

With the introduction of new regime laws, it is necessary to use semantic codes in their letter and spirit.
