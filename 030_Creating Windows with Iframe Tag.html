How to use 'base' and 'iframe' Tag

We can create new window with the use of iframe tag.

All types of files, i.e. jpg, png, gif, docx, pdf & html etc., can be made visible in new frame.

To display an image, use 'img src' attribute, the value of which is the full path of that image.

To display a file or directory, use 'src' attribute, the value of which is the full path of that file or directory.

Always use 'base' with attribute 'href' to shorten the path of a file.

The width and height can be adjusted of an iframe by using these attributes.





Techniques to use iframe Tag

There are three techniques to use iframe Tag.

The first method is to make a hyperlink of a file or image and target it to the desired iframe.

To call that file or image in the iframe, use 'name' or 'id' attribute inside iframe tag.
