Create Fields with Labels

A field as well as a button can be labeled by using 'label' tag.

At first, write the title of a field within 'label' tag.

Thereafter, use attribute 'for' within opening label tag and provide to attribute 'for' a suitable title.

In 'input' tag, use attribute 'id' and provide it the same title as of attribute 'for'.

How to create Buttons with Labels

A button can be created in HTML by using 'button' tag.

Alternatively, a button can also be created by using 'input' element.

The types are 'reset', 'submit', 'cancel', 'demo' and 'image' etc.

Attribute 'name' in case of 'button' tag and attribute 'value' in case of 'input' element must be used.

Attribute 'onclick' with value 'alert' may be used to display messages.

A button may be made disable by using attribute 'disabled'.

A short-key may be associated with a button by using attribute 'accesskey' with its value.

To call a button, press alt plus suggested key.

How to create Dialog Box

A dialog box can be created by using 'dialog' tag.

It can be linked to open further inside dialog boxes.

Use 'dialog' tag to create pop-up button with attribute 'open'.
