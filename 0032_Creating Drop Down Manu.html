Creating Drop Down Menu

We have three techniques to create drop down Menu.

First Method

Use 'select' tag, and inside it, use 'option' tag.

If we use this technique, first option automatically remain visible.

DOC. TYPE:

Second Method

Use attribute 'list' for input tag and pass an id to 'datalist' tag, thereafter use option tag within datalist tag. In that case, attribute 'list' and 'id' must be same.

If we use this technique, when we move our curser onto that field, drop down list will appear.

COUNTRY:

Third Method

If we have to choose a number of different items under different catagories, we use 'select' tag.

To do so, an attribute 'multiple' is used inside 'select' tag.

To make different groups, we use 'optgroup' tag within 'select' tag, and inside it, 'option' tag.

To highligh group's name, we use a attribute 'label' 'inside optgroup' tag.

To disable an option tag or optgroup tag, we use attribute 'disabled' in that tag.

To select the multiple options, always use 'ctrl' button.
