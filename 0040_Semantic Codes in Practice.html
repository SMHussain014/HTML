What is Meta Tag

This tag is semantic in nature and always used in 'head' tag.

Although it is not VISIBLE on web-page, but its result matters.

It made easy for the browser to search and locate a web-page during optimization.

It contains descriptive summary of a website, like defining keywords about data, author etc.

Usually, after the name of web-site, first two lines gives purpose, keywords for search & info about author of that web-site.

How to use Meta Tag

In VSC, if you type '!' and press tab or enter, basic html page appears automatically.

Meta:utf, and Meta:viewpoint will be appeared by default.

Meta: utf-8 tells about coding language and Meta:viewpoint tells about width=device-width, i.e. device size and initial-scale=1.1, i.e. initial viewable area.

You can also have some meta tags, like Meta:description (gives description and content), Meta:refresh (gives auto refresh property), Meta:property(og:image, og:title, og:description) etc.

This information will not be got printed but is visible in search engine, like google.

How to use Title, Content and Section Tags

These tags are semantic in nature and are not VISIBLE on web-page.

Section tag defines a group of co-related information to be used by search engine.

Title tag is used to define the objectives of an article as well as keywords for search etc.

Content tag contain the content of an article.

They made easy for the browser to search and locate an article during optimization.

<h1>Computer Network</h1>

A computer network is a set of computers sharing resources located on or provided by network nodes. Computers use common communication protocols over digital interconnections to communicate with each other. These interconnections are made up of telecommunication network technologies based on physically wired, optical, and wireless radio-frequency methods that may be arranged in a variety of network topologies.
<h1>Network Addresses</h1>

The nodes of a computer network can include personal computers, servers, networking hardware, or other specialized or general-purpose hosts. They are identified by network addresses and may have host names. Host names serve as memorable labels for the nodes and are rarely changed after initial assignment. Network addresses serve for locating and identifying the nodes by communication protocols such as the Internet Protocol.
