Hyperlink

A hyperlink or simply link refers to navigate, jump or move from one place to another.

To create a hyperlink or link, we have an element called 'Anchor' Tag.

The following are the applications of a hyperlink:

  1. to move from a website to another website;
  2. to join various web-pages of a website with each and other;
  3. to jump from one section of a webpage to other section;
  4. to display an image, an Audio Recording, a Video clip etc.

Types of Hyperlink

There are two primary types, a link has have:

  1. Absolute Link
  2. Relative Link

Absolute Link

This link is used to navigate from one website to another website.

To navigate, we need the address or domain name of a website which is called "url"

Domain name refers to a distinct and separate website.

Every website has its own server.

Uniform Resource Locater (URL) consists of four main parts:

  1. Scheme
  2. Server Name
  3. Path; and
  4. Resource

To understand the concept, we consider the following example:
https:\www.piaic.org\docs\entrytestkarachi

  • Scheme = http:
  • Server Name = www.piaic.org
  • Path = docs;
  • Resource = entrytestkarachi

Relative Link

This link is used to navigate within one website.

Actually, this type of link enables us to link all the webpages of a web-site.

To move through different webpages or sections of a website, we need a directory structure.

The directory structure may be understood with the example of a family tree.

Directory is simply a container where we keep our all the data.

Designing 'Directory Structure' marks the beginning of a project.

The main directory, where all the data is kept, is called 'Root Directory'.

A root directory may contains sub-directories, files, images, audios, videos etc.
