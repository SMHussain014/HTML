Use of Input Tag

We already know that all the data is entered by the use of empty element called 'input'.

Remember that 'input' tag works only within 'form' tag.

Types of Data

Now, we are about to discuss all the types of data which we may require from an user.

Data Type - Text

This type is used for writing alphanumeric materials in single line.

It has two semantic attributes, namely 'name' and 'value'.

It has three more attributes, namely 'size', 'maxlength' and 'autofocus'.

Attribute 'name' tells the browser about the purpose of data.

Attribute 'value' has tow aspects, i.e. first facilitate the browser how to react and second tells the user what to write.

Attribute 'size' fix the size of input field, yet it may be done by using CSS now.

Attribute 'maxlength' fix the length of characters to be entered by user.

Attribute 'autofocus' focus the input field when a form load.

If we want a field mandatory, use 'required' attribute.

Data Type - Textarea

This type is used for writing alphanumeric materials in multiple lines. It has opening as well as closing tags.

It has three attributes, i.e. cols, rows and placeholder.

Attributes 'cols' and 'rows' used to specify the width and height of textarea field, yet yet it may be done by using CSS now.

Attribute 'placeholder' is used to write a message by the user.

If attribute 'placeholder' is left empty no message will be shown.

If attribute 'placeholder' is set for readonly, in that case, the user is unable to erase data inside it.

If attribute 'placeholder' is set for disabled, in that case, cursor has no effect on this area.
