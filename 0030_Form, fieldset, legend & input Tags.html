How to create a Form

To create a form use form tag.

Within form tag, use its attributes, i.e. action and method.

Between form tag, use empty element 'input'. It has three attributes, i.e. type, name & value.

Attribute 'type' specifies the type of data needs to be entered by the user.

Attribute 'name' tells the browser that what is the purpose of this input data. It will not display on web-page being semantic in nature.

Attribute 'value' is used to (i) give description for user and (ii) provide real-time value for browser indicating how to react.

How to introduce sections in a Form

To divide a Form into section or parts, we have 'fieldset' tag.

To give name or title to a particular section of a from use 'legend' tag within 'fieldset' tag.

In earlier versions of HTML, we used width and height attributes for form but this will be done by using CSS now.
