What is Meta Tag

This tag is semantic in nature and always used in 'head' tag.

Although it is not VISIBLE on web-page, but its result matters.

It made easy for the browser to search and locate a web-page during optimization.

It contains descriptive summary of a website, like defining keywords about data, author etc.

Usually, after the name of web-site, first two lines gives purpose, keywords for search & info about author of that web-site.

How to use Meta Tag

In VSC, if you type '!' and press tab or enter, basic html page appears automatically.

Meta:utf, and Meta:viewpoint will be appeared by default.

Meta: utf-8 tells about coding language and Meta:viewpoint tells about width=device-width, i.e. device size and initial-scale=1.1, i.e. initial viewable area.

You can also have some meta tags, like Meta:description (gives description and content), Meta:refresh (gives auto refresh property), Meta:property(og:image, og:title, og:description) etc.

This information will not be got printed but is visible in search engine, like google.
