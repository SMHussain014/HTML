Absolute Link

This link is used to navigate from one website to another website.

To navigate from one website to another, we need a tag called "Anchor Tag".

Anchor Tag

This tag works with an attribute, namely 'href'.

The syntax of this tag is: opening tag: a href="URL", between tags: write Link Text, and closing tag.

Here 'href' acts as a target link for that website where you want to jump or move.

Its value is 'Link Text', present on our webpage, which acts as a hyperlink.

Link text should be relevant, i.e. indicating the exact name of website where you desire to move on.
