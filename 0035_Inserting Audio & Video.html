How to use Audio Tag

An audio can be inserted in the web-page by using 'audio' tag.

To insert an audio, we have an empty element 'source' which is used inside 'audio' tag.

The 'source' tag has two attributes, i.e. src (the value of which is complete path and desired file_name with extension) and type (the value of which is type of audio with extension, separated by forward slash).

Audio Tag has certain attributes, i.e. controls, preload, muted, and autoplay.

Attribute 'controls' display the audio on screen.

Attribute 'preload' controls the control panel of audio. Its value should be 'audio', if it is set to 'none' control panel becomes disabled.

Attribute 'loop' repeats the audio time & again.

Attribute 'autoplay' runs the audio automatically as the webpage loads.

Attribute 'muted' is used to mute the audio.

How to use Video Tag

A Video can also be inserted in the web-page as of audio. The only difference is the use of 'video' tag.

All the attributes used for 'audio' tag may be used for 'video' tag.

In case of 'video' tag, we have three more attributes, i.e. width, height and poster.

We know how to use width & height attributes. Attribute 'poster' demands full path like image.

How to use Object Tag

If an audio or video file has same name, we can use 'object' tag also.

To provide path, use 'data' attribute and to specify file type, use 'type' attribute.

All the attributes used for audio and video tags may be used for object tag.

If we want to give name that file, use attribute 'name'.

Other than that object tag has also an inline tag called 'param'.

the element 'param' has three attributes namely name, value and style.

The attribute 'value' has only two values, i.e. true or false, which is used to run an audio or a video on autoplay mode.

If browser does not support 'param' inline tag, use attribute 'style', like style="display:none;"

How to use Embed Tag

This tag can be used like 'object' tag.

The only difference is use of 'src' attribute instead of 'data' attribute.
