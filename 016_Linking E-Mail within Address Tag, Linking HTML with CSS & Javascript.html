How to link E-Mail

E-mail can also be used as a hyperlink.

To do this: use anchor tag's attribute 'href' with another attribute 'mailto'.

The syntax is, open anchor tag, use attribute href, assign its value mailto:full_e-mail_address, write text link between both tags, close anchor tag.

Script and Noscript Tags

This tag is used, either in head or body, to create a link between HTML & JavaScript.

We have two techniques to use client side script, i.e. in-page script & external script.

It has four attributes, namely type, src, async and defer.

Attribute 'type' tells the browser what type of programming language is to be used.

Attribute 'src' provides the path from where a programming file exists.

Attributes 'async' and 'defer' provides the method how to execute the programming language.

If method 'async' is used, the web-site will be loaded first and then the programming language.

The method 'defer' reverse the above process.

If browser does not support a programming language, then noscript tag will be executed.
