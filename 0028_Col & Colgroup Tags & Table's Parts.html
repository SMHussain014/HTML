Parts of Tables

A table has three parts, i.e. head, body and food.

All these parts can be separated from each and other.

For this, we have the following tags:

  • thead
  • tbody
  • tfoot

These tags are semantic and browser will not show their result.

How to use col and colgroup tags

Both 'col' tag and 'colgroup' tag used to work with columns

Both acts the same. Their attributes are 'span' and 'style'.

span attribute refer to number of columns and style attribute does the rest.

Tag 'colgroup' can be used with or without col tag.

The col tag is an empty element, i.e. have no closing tag.
