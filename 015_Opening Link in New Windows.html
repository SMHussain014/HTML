Opening Link in New Windows

Anchor tag is used to navigate both, i.e within one website, and to other website.

This tag is used with attribute 'href'.

Attribute "_self"

Default attribute of this Tag is '_self', meaning thereby windows open in its own frame.

Attribute "_target"

The value of 'target' attribute tell the browser where to open a website. To open in new Windows, its value should be _blank.

Attribute "title"

The value of 'title' attribute, shows the message for the user, when you place curser on it.
