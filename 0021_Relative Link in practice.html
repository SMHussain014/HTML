Relative Link

This link is used to navigate within one website and joins all the webpages of that website.

To navigate from one page to other, we need a tag called "Anchor Tag".

Anchor Tag

This tag is used with an attribute, namely 'href'.

The syntax of this tag is: opening tag: a href="Path/file_name.html", between tags: write Link Text, and closing tag.
