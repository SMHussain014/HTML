How Summary and Details Tags work

These tags are used in co-relation to each and other.

In HTML, 'summary' tag is used to show summary of anything.

This tag is always used within 'details' tag.

In HTML, 'details' tag is used to hide the details from the user for the time being.

It has only one attribute namely 'open'. if used, shows the details all the time.

How to use Data Tag

In business, if we call an item by its name, it becomes quite difficult.

On the other hand, we want to hide this information for the end user.

In that case, we use 'data' tag which enables us to call items by code and hide these codes from user.

To do so, data tag has one attribute 'value'.
